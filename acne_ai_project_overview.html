🧬 ACNE AI

An AI-powered acne severity analysis platform that uses deep learning to classify skin conditions and provide personalized skincare recommendations.

🏗️ What We Built (From Scratch)

The Problem

People struggle to objectively assess their acne severity and don't know what products or routines are right for their skin type. Dermatologist visits are expensive and inaccessible for many.

The Solution

A web application where users:

  1. Upload or capture a photo of their face
  2. Get an instant AI-powered severity diagnosis (Mild / Moderate / Severe)
  3. Receive a personalized skincare plan including morning/night routines, product recommendations, do's/don'ts, and treatment timelines — all tailored to their skin type

🧠 AI / ML Pipeline — How It Works

📸 User uploads photo → 🔍 Face Detection (MediaPipe) → ✂️ Crop Face Region → 🧠 Severity Classifier (EfficientNetB3) → 📊 Confidence Scoring → 💊 Personalized Recommendations → 📋 Full Report

Step-by-Step

  1. Image Input — User uploads a photo or captures one via webcam
  2. Face Detection — MediaPipe detects and crops the face region. Non-face images are rejected
  3. Preprocessing — Face is resized to 300×300 pixels for the CNN
  4. Severity Classification — EfficientNetB3 classifies into 3 classes:
    • Mild — Blackheads, whiteheads (comedonal acne)
    • Moderate — Papules, pustules (inflammatory acne)
    • Severe — Cysts, nodules (nodulocystic acne)
  5. Confidence Check — If confidence < 40%, result is marked "Inconclusive"
  6. Recommendations Engine — Based on severity + user's skin type, generates personalized routines, product recs, timelines

📊 Model Training Details

Model Architecture

EfficientNetB3 (ImageNet pre-trained)
├── Data Augmentation (RandomFlip, RandomRotation, RandomZoom, RandomContrast)
├── EfficientNetB3 Backbone (frozen → fine-tuned)
├── GlobalAveragePooling2D
├── BatchNormalization + Dropout (0.4)
├── Dense(512, ReLU) + BatchNorm + Dropout (0.3)
├── Dense(256, ReLU) + BatchNorm + Dropout (0.2)
└── Dense(3, Softmax) → [Mild, Moderate, Severe]

Training Strategy (2-Phase Transfer Learning)

PhaseWhat HappensEpochsLearning Rate
Phase 1: Feature ExtractionBackbone frozen, only head trains151e-6 → 1e-3 (cosine warmup)
Phase 2: Fine-TuningTop 50% of backbone unfrozenUp to 601e-7 → 3e-5 (cosine annealing)

Training Techniques

  • Transfer Learning — Pre-trained EfficientNetB3 (ImageNet weights)
  • Cosine Annealing with Warmup — Custom LR scheduler
  • Label Smoothing (0.1) — Prevents overconfident predictions
  • Class Weighting — Balanced weights for imbalanced severity classes
  • Early Stopping — Stops when validation loss plateaus (patience: 15)
  • Data Augmentation — Random flips, rotations, zoom, contrast
  • tf.data.Dataset — Fast multi-threaded data loading with AUTOTUNE prefetching

🔑 Key Technical Decisions

  1. EfficientNetB3 over MobileNetV2 — Better accuracy-to-parameter ratio for medical imaging
  2. Keras 3 + PyTorch backend — Best of both worlds: Keras simplicity + PyTorch GPU speed
  3. tf.data.Dataset — 10x faster than ImageDataGenerator for large datasets
  4. 2-Phase Training — Phase 1 learns acne patterns, Phase 2 fine-tunes for skin-specific details
  5. MediaPipe Face Detection — Lightweight, accurate, rejects non-face uploads
  6. Personalized Recommendations — Not just diagnosis, but actionable skincare plans by skin type
