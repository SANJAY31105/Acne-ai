💊 ACNE AI — Treatment Recommendation System

Complete technical documentation of how the personalized skincare recommendation engine works — from algorithm to clinical sources.

1. System Architecture Overview

The Acne AI app uses a two-layer architecture that separates diagnosis from treatment:

📸 User Photo
→
🧠 Layer 1: AI Model
EfficientNetB3 CNN
Output: Mild / Moderate / Severe
→
📝 Skin Quiz
Oily / Dry / Combo / Sensitive / Normal
→
💊 Layer 2: Rule-Based Engine
Lookup: Severity × Skin Type
Source: recommendations.json
→
📋 Personalized Report

Key Design Decision: The AI model performs ONLY the diagnosis (classification). The treatment recommendations come from a curated, static clinical knowledge base — NOT from AI generation. This ensures safety, auditability, and determinism in medical advice.

2. Algorithm Used

For Diagnosis → Deep Learning (EfficientNetB3 CNN)

A Convolutional Neural Network classifies the face image into 3 severity classes using transfer learning from ImageNet.

For Recommendations → Rule-Based Expert System

A deterministic lookup table (JSON database) maps [Severity × Skin Type] to a complete treatment plan. This is the same approach used in clinical decision support systems (CDSS) in hospitals.

Approach Used For Why
Deep Learning (CNN) Image → Severity classification Learns visual patterns from pixels automatically
Rule-Based Expert System Severity + Skin Type → Treatment plan Safe, deterministic, auditable for medical use

5. Severity-Specific Treatment Logic

MILD — Comedonal Acne (Blackheads & Whiteheads)

Clinical Logic: Mild acne = clogged pores, NOT bacterial infection. Pores are blocked by dead skin cells and sebum (oil). Treatment focuses on chemical exfoliation to dissolve the buildup and retinoids to speed skin cell turnover.

Ingredient Mechanism Evidence
Salicylic Acid (BHA) 2% Oil-soluble acid penetrates pores, dissolves keratin plugs AAD first-line OTC for comedonal acne
Niacinamide (Vitamin B3) Reduces sebum production by 23% (4 weeks), anti-inflammatory Journal of Dermatological Treatment, 2013
Retinol / Adapalene Increases skin cell turnover, prevents new comedones from forming FDA-approved OTC retinoid (Differin)

MODERATE — Inflammatory Acne (Papules & Pustules)

Clinical Logic: Moderate acne = Cutibacterium acnes (C. acnes) bacteria actively infecting clogged pores, causing inflammation. Treatment requires antibacterial agents (BP) + retinoids (Adapalene) on separate schedules to avoid deactivation.

Ingredient Mechanism Evidence
Benzoyl Peroxide 2.5-5% Releases oxygen inside pores, killing anaerobic C. acnes bacteria AAD + NICE guidelines: first-line antibacterial
Adapalene (Retinoid) 0.1% Normalizes follicular keratinization, anti-inflammatory FDA-approved, 50%+ lesion reduction in clinical trials
Azelaic Acid 15% Dual: antibacterial + reduces post-inflammatory marks EU-approved for acne and rosacea
Hydrocolloid patches Absorbs pus, creates moist healing environment, prevents picking Standard wound care science

Important Rule: BP (morning) and Adapalene (night) are NEVER used at the same time — BP oxidizes retinoids and deactivates them. This is reflected in the separate AM/PM routines.

SEVERE — Nodulocystic Acne (Cysts & Nodules)

Clinical Logic: Severe acne = deep infections beneath the skin surface. OTC products cannot physically reach these lesions. Every treatment plan explicitly states: "See a dermatologist — prescription treatment is essential."

Treatment Mechanism Evidence
Isotretinoin (Accutane) Shrinks sebaceous glands by 80%, permanently reduces oil production Gold standard: 80% long-term remission (NEJM)
Doxycycline (oral antibiotic) Systemically reduces bacterial load during active flares AAD first-line oral antibiotic for acne
Spironolactone Anti-androgen — blocks hormonal triggers (women only) Effective for hormonal acne (JAAD, 2017)
Cortisone injection Intralesional steroid flattens cysts within 24-48 hours Standard dermatology in-office procedure

Safety Note: For Severe classification, our app recommends ONLY ultra-gentle OTC products (Vanicream, CeraVe, SPF) alongside explicit instructions to see a professional. We never recommend prescription drugs without a dermatologist.

6. How Skin Type Modifies Treatment

The same severity gets different products based on skin type. Here's an example for MODERATE acne:

🫧 Oily Skin

  • BP 5% wash (higher concentration OK)
  • Gel-based, oil-free moisturizer
  • Zinc supplements (30mg/day) to reduce oil

🏜️ Dry Skin

  • BP 2.5% only (lower to avoid drying)
  • Hyaluronic Acid serum before moisturizer
  • Squalane oil (non-comedogenic hydration)
  • Cream cleanser, NOT foaming

⚠️ Sensitive Skin

  • Skip BP entirely → Azelaic Acid 15% instead
  • Retinoid only 2-3x/week at lowest dose
  • Aloe Vera gel as moisturizer
  • Fragrance-free everything

🔄 Combination Skin

  • BP as spot treatment (inflamed areas only)
  • Retinoid on oily/acne zones, skip dry areas
  • Niacinamide balances oil naturally

Why this matters: Using BP 5% on sensitive skin causes chemical burns. Using cream moisturizer on oily skin clogs pores further. The skin type layer ensures products are safe AND effective for each user.

7. Code Implementation

Data Storage: recommendations.json

A 375-line JSON file storing all treatment data. Structure:

{
  "Mild": {
    "likely_types": ["Blackheads", "Whiteheads"],
    "timeline": "4-6 weeks",
    "key_ingredients": [{ "name": "Salicylic Acid 2%", "purpose": "Unclogs pores" }, ...],
    "morning_routine": ["SA cleanser", "Niacinamide serum", "SPF moisturizer"],
    "night_routine": ["Double cleanse", "BHA 3×/week", "Retinol", "Ceramide cream"],
    "dos": ["Change pillowcases", "2L water/day", ...],
    "donts": ["Don't squeeze", "Don't skip SPF", ...],
    "products": [{ "name": "CeraVe SA Cleanser", "type": "Cleanser", "key_ingredient": "SA 2%" }],
    "by_skin_type": {
      "Oily": ["Use oil-free SA cleanser...", "Clay masks weekly..."],
      "Dry": ["Use cream-based cleanser...", "HA serum..."],
      "Sensitive": ["Avoid SA, use Azelaic Acid 10%...", "Fragrance-free..."],
      ...
    },
    "type_treatments": {
      "Blackheads": ["BHA dissolves pore buildup...", "Retinol prevents new ones..."],
      "Whiteheads": ["Adapalene speeds cell turnover...", "Never squeeze..."]
    }
  },
  "Moderate": { ... },
  "Severe": { ... },
  "Inconclusive": { ... }
}

Lookup Function: pipeline.py → get_recommendations()

def get_recommendations(self, severity, skin_type="Normal"):
    # 1. Load the clinical knowledge base
    recommendations_db = json.load(open("config/recommendations.json"))

    # 2. Look up by AI-predicted severity
    entry = recommendations_db[severity]   # e.g. "Moderate"

    # 3. Get skin-type-specific advice from user quiz
    skin_recs = entry["by_skin_type"].get(skin_type, entry.get("general", []))

    # 4. Return complete personalized report
    return {
        "likely_types":      entry["likely_types"],
        "key_ingredients":   entry["key_ingredients"],
        "morning_routine":   entry["morning_routine"],
        "night_routine":     entry["night_routine"],
        "dos":               entry["dos"],
        "donts":             entry["donts"],
        "products":          entry["products"],
        "timeline":          entry["timeline"],
        "personalized_plan": skin_recs,         # ← skin-type specific
        "type_treatments":   entry["type_treatments"]
    }

9. Safety Measures

Safety Feature How It Works
Confidence Threshold If AI confidence < 40%, result="Inconclusive" — no treatment is recommended
Face Detection Gate MediaPipe rejects non-face images before any classification happens
No AI-generated treatments All recommendations are from a static, auditable JSON database — zero hallucination risk
Severe = See Doctor Every Severe treatment plan mandates dermatologist consultation
Skin type filtering Prevents dangerous ingredient mismatches (e.g., BP on sensitive skin)
Deterministic output Same input = same output every time. No randomness.

10. How to Explain This in a Presentation

One-liner:

"Our AI diagnoses the severity using a deep learning CNN, and then a rule-based expert system built on AAD clinical guidelines gives personalized treatment recommendations based on both the severity and the user's skin type."

If asked "Why not use AI for recommendations too?":

"We deliberately chose a rule-based approach for safety. AI-generated medical advice can hallucinate or give wrong products for a specific skin type. A static, expert-reviewed database is more reliable, explainable, and safe for a healthcare application."

If asked "Where did the treatments come from?":

"Every ingredient is backed by published clinical evidence. Benzoyl Peroxide and Adapalene are FDA-approved. Isotretinoin has an 80% remission rate per the NEJM. The recommendations follow AAD guidelines for acne management."
